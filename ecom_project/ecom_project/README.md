# 🛒 E-Commerce Data Analysis Project

A complete, beginner-friendly data analysis project for Data Science Learners.
Includes a 500-row Kaggle-style dataset, Python scripts, Excel report, and
an interactive light-theme analytics dashboard.

---

## 📁 Folder Structure

```
ecom_project/
│
├── data/
│   ├── orders.csv          ← Dataset (500 orders, Kaggle-style, Jan 2023–Dec 2024)
│   └── orders_clean.csv    ← Auto-generated after running Script 01
│
├── scripts/
│   ├── 01_analysis.py      ← Full analysis: 10 steps, KPIs, groupby aggregations
│   └── 02_excel_report.py  ← Auto-generates Excel report (5 sheets)
│
├── dashboard/
│   └── index.html          ← Interactive light-theme dashboard (open in browser)
│
├── outputs/
│   └── ecommerce_report.xlsx   ← Generated by Script 02
│
└── README.md
```

---

## 🚀 How to Run (Step by Step)

### 1. Install Python
Download from https://python.org/downloads
During install → tick ✅ "Add Python to PATH"

### 2. Install Libraries
Open terminal INSIDE the ecom_project folder, then run:
```
python -m pip install pandas numpy openpyxl
```

### 3. Open Terminal Inside This Folder
Windows: Open the folder in File Explorer → click the address bar → type cmd → press Enter
Mac: Right-click the folder → "New Terminal at Folder"

### 4. Run the Analysis Script
```
python scripts/01_analysis.py
```
Prints 10-step analysis in the terminal. Saves data/orders_clean.csv.

### 5. Generate the Excel Report
```
python scripts/02_excel_report.py
```
Creates outputs/ecommerce_report.xlsx with 5 formatted sheets.

### 6. Open the Dashboard
No terminal needed. Just double-click dashboard/index.html in File Explorer.
Opens in Chrome, Firefox, or Edge. Works offline, no internet needed.

## 📊 Dataset Columns

| Column         | Description                                    |
|----------------|------------------------------------------------|
| order_id       | Unique transaction ID (starts at 10001)        |
| order_date     | Purchase date (YYYY-MM-DD)                     |
| ship_date      | Delivery date                                  |
| ship_mode      | Standard / Express / Same Day                  |
| customer_id    | Customer code (C101–C120)                      |
| customer_name  | Full name of the buyer                         |
| region         | North / South / East / West / Central          |
| category       | Electronics, Clothing, Books, Beauty, etc.     |
| product_name   | Item purchased                                 |
| quantity       | Units bought                                   |
| unit_price     | Price per unit before discount                 |
| discount_pct   | Discount % applied (0, 5, 10, 15, or 20)       |
| revenue        | quantity × unit_price (before discount)        |
| discount_amt   | Rupee value of discount                        |
| net_revenue    | Final amount received (after discount)         |
| payment_method | Credit Card / UPI / Debit Card / Net Banking   |
| returned       | Yes or No                                      |
| rating         | Customer rating 1–5                            |

Visit one of these Kaggle datasets (free account needed):
